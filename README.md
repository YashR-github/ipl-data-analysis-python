# IPL Data Analysis using Python

This project aims to extract, analyze and visualize varied and meaningful information from the IPL 2008-2016 csv datasets using Python, Pandas and Matplotlib. 


# Steps to follow:

## 1. Ensure Python Runtime Exist

Run the following command in terminal to check whether Python is installed and which version is it. 

```
python3 --version
```

If Python is not installed, run the following commands to install latest version of Python from terminal.

```
sudo apt update
sudo apt install python3
```

## 2. Install pip - Python's package installer

pip is used to install Python Packages like Pandas, Matplotlib etc. To install, run the following commands in the terminal after installing Python.

```
sudo apt install python3-pip
```
To check whether pip is installed correctly, run:

```
pip3 --version
```

## 3. Install pandas and matplotlib

Run the following in terminal to install pandas and matplotlib which are required in order to run the code.

```
pip3 install pandas matplotlib
```

Alternatively, you can also run administrative command to allow permission to install in case of any error received in above command:

```
sudo apt install python3-pandas matplotlib
```
Note: This command requires admin credentials in order to run.

## 4. Clone the repository

Finally, clone this repository in your local environment to run and receive output as bar graphs in png format in the 'outputs' directory.
